# CPPL – Inbound Process Automation

**Client:** Corrosion Protection Private Limited (CPPL)  
**Developed by:** Webteam Private Limited (in partnership with TransExcel Consulting)  
**Project Start:** February 2026

---

## Project Overview

A purpose-built web-based Inbound Automation System replacing manual registers and WhatsApp-based workflows at CPPL. The system covers end-to-end material flow from procurement to scrap reuse, with bidirectional integration to Tally ERP and the existing cutting machine software.

## Key Integrations

- **Tally ERP** – Bidirectional (GRN, inventory, financial entries)
- **Cutting Machine Software** – Import job data (CSV / API / DB), auto-deduct inventory and auto-create scrap entries

---

## Infrastructure

- Cloud-hosted on AWS / Azure
- 99.5% uptime SLA, daily automated backups
- Role-based access control (RBAC), audit logs, data encryption at rest & in transit
